```
DISCORD_BOT_TOKEN=your_discord_bot_token
TEXTRAZOR_API_KEY=your_textrazor_api_key
GIPHY_API_KEY=your_giphy_api_key
TWITCH_CLIENT_ID=your_twitch_client_id

```

